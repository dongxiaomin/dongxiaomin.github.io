Jquery

2020-05-07

一. 概述

jQuery 是一个高效、精简并且功能丰富的 JavaScript 工具库。它提供的 API 易于使用且兼容众多浏览器，这让诸如 HTML 文档遍历和操作、事件处理、动画和 Ajax操作更加简单。

  • 全面解决pc端的兼容性问题。
  • 语法精炼，性能好。插件库非常庞大。

二. 下载 jQuery

  1. 打开cdn链接
  2. 搜索 jquery
  3. 复制 <script> 标签

    <script src="https://cdn.bootcdn.net/ajax/libs/jquery/3.5.0/jquery.js"></script>

三. jQuery库包含以下功能：

  1. HTML 元素选取
  2. HTML 元素操作
  3. CSS 操作
  4. HTML 事件函数
  5. JavaScript 特效和动画
  6. HTML DOM 遍历和修改
  7. AJAX
  8. Utilities
    提示： 除此之外，Jquery还提供了大量的插件。
